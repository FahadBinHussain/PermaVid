PermaVid Index Server API

This is the API server for the PermaVid public index. The following endpoints are available:

POST /add

Add a video URL to the index.

Body: { "url": "https://youtube.com/watch?v=example" }

GET /check?url=...

Check if a video URL exists in the index.

Response: { "found": true/false }

For more information, see the GitHub repository.
